# ITUNE-MUSIC-SALES-DA
## 📌 Project Overview

This project analyzes iTunes music sales data using SQL and Power BI to identify key business insights such as revenue distribution, customer purchasing behavior, top-performing genres, and geographic sales performance.

The dashboard helps stakeholders understand which music genres generate the most revenue, which countries contribute the most sales, and how customer purchasing patterns affect overall revenue.

## 🎯 Business Problem

Music platforms generate large volumes of transactional data.
The objective of this project is to analyze sales data to answer the following business questions:

Which music genres generate the highest revenue?

Which countries generate the most revenue?

How do customer purchasing behaviors impact sales?

How do sales vary across quarters?

Which genres have the highest number of tracks sold?
